Personalized Financial Advisor using Large Language Model (LLM)

The field of finance can be complex and overwhelming for individuals seeking personalised financial advice. In order to make informed decisions regarding investments, retirement planning, budgeting, and financial products, individuals often require guidance from financial experts. The aim of this project is to develop an Intelligent Financial Advisor powered by a Large Language Model (LLM) to provide personalised financial advice and guidance to individuals. By leveraging NLP and machine learning techniques, the Intelligent Financial Advisor will assist users in making informed financial decisions and achieving their financial goals.

This complete analyser is based in NSE data, which uses LSTM for prediction based on past prices and LLM for summarizing and sentiment analysis of news.
